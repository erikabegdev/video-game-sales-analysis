# 🎮 Análise de Vendas de Videogames

## 📌 Objetivo

Este projeto tem como objetivo analisar dados históricos de vendas de jogos eletrônicos para identificar padrões que determinam o sucesso comercial no mercado de games e apoiar decisões estratégicas para o ano de 2017.

A análise considera fatores como plataforma, gênero, avaliações de usuários e críticos, classificação etária (ESRB) e diferenças regionais de consumo.

---

## 🛠️ Habilidades Aplicadas

- Python  
- Pandas  
- Limpeza e pré-processamento de dados  
- Análise Exploratória de Dados (EDA)  
- Visualização de dados (Matplotlib e Seaborn)  
- Testes estatísticos (teste t para amostras independentes)  
- Interpretação de resultados e geração de insights  

---

## 📊 Principais Análises Realizadas

- Evolução do número de jogos lançados ao longo dos anos  
- Identificação das plataformas com maior volume de vendas no período recente  
- Distribuição de vendas por plataforma (boxplot)  
- Análise da relação entre avaliações e desempenho comercial  
- Identificação dos gêneros mais lucrativos  
- Perfil de consumo por região (América do Norte, Europa e Japão)  
- Testes de hipóteses para comparação de médias  

---

## 📈 Principais Resultados

- O ciclo de vida das plataformas impacta diretamente o volume de vendas.  
- Existem diferenças regionais significativas nas preferências por plataforma e gênero.  
- Não foi identificada diferença estatisticamente significativa entre as avaliações médias de Xbox One e PC.  
- Foi identificada diferença estatisticamente significativa entre as avaliações médias dos gêneros Action e Sports.  

---

## 🎯 Conclusão

O sucesso de um jogo é determinado por múltiplos fatores combinados, como plataforma, gênero, preferências regionais e momento de mercado.

Os resultados reforçam a importância da análise de dados na tomada de decisão estratégica e demonstram como técnicas de EDA e testes estatísticos podem apoiar decisões de negócio.
