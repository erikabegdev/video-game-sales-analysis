## Video Game Sales Analysis
### Objective

** Analyze historical video game sales data to identify patterns that determine market success and support strategic decisions for 2017.** 

- Skills Applied
- Python
- Pandas
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Hypothesis Testing
- Key Findings
- Platform lifecycle impacts sales trends
- Significant regional differences in preferences
- No significant difference between Xbox One and PC user ratings
- Significant difference between Action and Sports genres
